# While You Were Busy

A social impact video game exploring AI's effect on the world and environment. Built as a game design capstone project with a team of three.

<!-- Add gameplay GIF or screenshot here: ![Gameplay](gameplay.png) -->

## About

**While You Were Busy** asks players to confront the hidden costs of AI — energy consumption, environmental impact, and societal change — through interactive storytelling and game mechanics.

*Built as a capstone project for Game Design, collaborating with a team of three.*

## My Role

- Unity development — built core game systems and mechanics
- Game design — designed gameplay loops, progression, and narrative structure
- UI design and implementation
