================================================================================
                        TROUBLESHOOTING GUIDE
================================================================================

PROBLEM: No tickets spawn
  CAUSE: TicketPrefab not assigned
  FIX: Drag TicketPrefab.prefab into Printer script field

PROBLEM: Tickets don't animate
  CAUSE: BulletinBoard not in scene or not found
  FIX: Create BulletinBoard GameObject with BulletinBoard script

PROBLEM: Wrong spawn position
  CAUSE: TicketSpawnPoint positioned incorrectly
  FIX: Select Printer → TicketSpawnPoint, adjust Anchored Position

PROBLEM: No sound on print
  CAUSE: Audio clip not assigned or AudioSource disabled
  FIX: Assign audio clip to AudioSource or enable it

For more details, see Assets/Scripts/PRINTER_SETUP_GUIDE.md
